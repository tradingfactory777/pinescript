프로젝트 3

볼린저밴드와 스토캐스틱을 활용한 자동 매매 시스템

프로젝트 개요

볼린저밴드와 스토캐스틱 오실레이터를 조합하여 시장의 변동성과 과매수/과매도 구간을 동시에 분석하고, 자동으로 매매를 실행하는 전략 스크립트입니다.

주요 기능

  • 볼린저밴드 이탈 및 회귀 시 자동 매매 신호 포착
  • 스토캐스틱 지표를 활용한 매매 시점 필터링
  • 백테스팅을 통한 전략 성과 분석 기능

개발 과정

과거 데이터를 기반으로 한 백테스팅을 통해 전략의 효율성을 검증하고, 파인스크립트의 전략 테스트 기능을 활용하여 최적의 매개변수 값을 찾아냈습니다.
